# Contributing

## Data corrections

Open a field-correction issue or pull request. Include the field name/code, what is wrong, and what source supports the correction.

## Photos

Only contribute photos you took yourself or have permission to contribute. Prefer useful cockpit material:

- approach from likely arrival directions
- final approach
- slope / obstacles / power lines
- runway or landable area condition
- windsock / landmarks

By default, new contributed photos should be under CC BY-SA 4.0. Do not submit copyrighted screenshots or documents unless redistribution is clearly allowed.

## Metadata

Strip EXIF metadata before publication. The build pipeline may do this later automatically; for now contributors should be conservative.

## Safety

This project is not authoritative. Corrections should make the app more useful, but the pilot remains responsible for current official information and safe decision-making.
